I.Miller-Rabin素性检验

对于奇素数 ppp, 当整数 aaa 与 ppp 互质时, 有费马小定理:

p−1p-1p−1 是偶数, 所以:

(ap−12)2≡1(modp)⇒(ap−12)2−1≡0(modp)⇒(ap−12+1)(ap−12−1)≡0(modp)⇒ap−12≡±1(modp)\begin{matrix} &&(a^{\frac{p-1}{2}})^2\equiv1 \pmod p\\ &\Rarr&(a^{\frac{p-1}{2}})^2-1\equiv0 \pmod p\\ &\Rarr&(a^{\frac{p-1}{2}}+1)(a^{\frac{p-1}{2}}-1)\equiv0 \pmod p\\ &\Rarr&a^{\frac{p-1}{2}}\equiv\pm1 \pmod p \end{matrix} ​⇒⇒⇒​(a2p−1​)2≡1(modp)(a2p−1​)2−1≡0(modp)(a2p−1​+1)(a2p−1​−1)≡0(modp)a2p−1​≡±1(modp)​

则当 ppp 是奇素数时, 同余方程 x2≡1(modp)x^2\equiv1\pmod px2≡1(modp) 的解只有 ±1\pm1±1 (称为平凡因子).

当 p=2p=2p=2 时, 可以得出方程 x2≡1(modp)x^2\equiv1 \pmod px2≡1(modp) 的解只有 111.

假设需要检验的数为 xxx.

当 xxx 为偶素数时, 仅有一种情况 x=2x=2x=2, 特判即可.

当 xxx 为奇素数时, 对于 a∈[1,x)∩Z+a\in[1,x)\cap\Z_+a∈[1,x)∩Z+​, 一定有费马小定理 ax−1≡1(modx)a^{x-1}\equiv1\pmod xax−1≡1(modx);

Jim Sinclair发现的数列 A={2,325,9375,28178,450775,9780504,1795265022}A=\{2, 325, 9375, 28178, 450775, 9780504, 1795265022\}A={2,325,9375,28178,450775,9780504,1795265022} 作底数时, 可保证 Miller-Rabin 在 2642^{64}264 内不会出错.

Code:

typedef long long ll;
ll qpow(int a,int e,int m)//快速幂, 计算a^e mod m
ll arr[7]={2,325,9375,28178,450775,9780504,1795265022};//用于探测的底数们
bool chkprm(ll x)//本体.
{
	if(x<3) {
        //特判:偶素数.
		return x==2;
	}
	if(x%2==0) {
        //特判:偶合数.
		return false;
	}
	ll d=x-1,r=0;
	while(d%2==0) {//分解x为2^r*d.
		d/=2;
		r++;
	}
	ll base;
	for(int i=0; i<7; i++) {
		base=qpow(arr[i],d,x);//base现记录a[i]^d. 即 a[i]^(2^0)*d
		if(base<=1||base==x-1) {
        /*如果base为0, x是a[i]的因数.
          如果base为1, 那么a[i]^(2^0)*d===1(mod x), 
          且接下来都是1, 属于情况(1).
          如果base为x-1, 那么a[i]^(2^0)*d===-1(mod x),
          且接下来都是1, 属于情况(2).*/
			continue;
		}
		for(int i=0; i<r; i++) {
			base=(__int128_t)base*base%x;
            //base自乘. 相当于计算数列的下一项.
			if(base==x-1&&i!=r-1) {
                //遇到了-1, 而且它不在结尾上. 那么接下来都是1, 属于情况(2).
				base=1;
                /*将base变为1(已知序列以1结尾), 再跳出 
                否则跳出后在46行的if处被误判为"以-1结尾".*/
				break;
			}
			if(base==1) {
                //遇到了1, 但是前面没有遇到-1(若经过-1则会在38行处跳出)
                //属于情况(3), 直接判定x非质数.
				return false;
			}
		}
		if(base!=1) {
            //结尾不是1. 属于情况(4), 判定x非质数.
			return false;
		}
	}
	return true;
}

II. Pollard-Rho算法分解质因数

如果需要找到 nnn 的一个因数, 试除法的期望复杂度大约是 O(n)O(\sqrt{n})O(n​), nnn 的规模过于夸张的话是不可接受的.

考虑用随机化提高试除的概率, 当选择一个数字 k∈Z+k\in\Z_+k∈Z+​, n,kn,kn,k 形成的最大公因数 (GCD) gcd⁡(n,k)\gcd(n,k)gcd(n,k) 只要不为 111 或 nnn ,就可以作为 nnn 的一个非平凡因数.

通过对家喻户晓的生日悖论进行研究, 可以发现, 选取数个随机数产生 nnn 的所有因数的概率很高.

Pollard设计了伪随机数列函数 f(x)=(x2+C)mod  nf(x)=(x^2+C)\mod nf(x)=(x2+C)modn, 其中 CCC 是一个随机生成的常数. Pollard-Rho 使用了该函数不断迭代(即将 f(x)f(x)f(x) 代入继续生成 f(f(x))f(f(x))f(f(x)), 如此类推)生成随机数, 并将相邻两项作差, 用这个差求 GCD.

f(x)f(x)f(x) 在迭代到一定次数后会进入循环. 类比一下希腊字母 ρ\rhoρ (rho, 算法名字的后半部分), 先生成一段数列, 而后生成内容被引向一个环.

此时寻找因数是没有意义的, 因此要引入判环算法帮助解决. Floyd 判环算法定义了一个乌龟变量和一个兔子变量, 分别记为 t,rt,rt,r. 每一轮迭代都使得 t←f(t),r←f(f(r))t\larr f(t), r\larr f(f(r))t←f(t),r←f(f(r)), 则如果进入环, rrr 和 ttt 的值会在某一点相遇, 于是我们更改常数 CCC, 重新寻找因数.

Code:

ll gcd(ll x,ll y)//返回x,y的最大公因数.
ll randint(ll L,ll r)//返回位于[L,R]的一个随机整数.
ll func(ll x,ll cst,ll mod)//f(x)函数, 参数cst表示常数C, 参数mod表示模数.
{
	return ((__int128_t)x*x+cst)%mod;
}
ll fact(ll x)//本体, 返回一个x的因数, 如果返回x, 说明没找到.
{
    ll rndnum=randint(1,x-1);//生成常数C!
    ll tur=0,rab=0;//乌龟变量, 兔子变量.
    while(t!=r)
    {
        ll _gcd=gcd(abs(tur-rab),x);//相邻两项作差, 求GCD.
        if(_gcd>1)
            return _gcd;
        tur=func(tur,rndnum,x),//乌龟跑一步.
        rab=func(func(rab,rndnum,x),rndnum,x);//兔子跑两步.
    }
    return x;//没找到, 返回重新生成常数C.
}

用辗转相除法求解 gcd⁡(x,y)\gcd(x,y)gcd(x,y) 的复杂度为 O(log⁡n)O(\log n)O(logn), 反复调用这个函数会使程序跑得很慢, 考虑到 GCD 可重复贡献的性质, 考虑将所有的 ∣t−r∣|t-r|∣t−r∣ 全部乘起来再求 GCD.

还有一个玄学的常数优化, 在迭代次数超过 128128128 而未退出环时考虑直接舍弃掉这个环, 直接生成新的常数参与运算.

Code:

ll fact(ll x)
{
	if(x==4)//特判4
	{
		return 2;
	}
	if(chkprm(x))//一定要先判断质数, 否则31处求得因数的值只能是1, 无法退出.
	{
		return x;
	}
	while(true)
	{
		ll rndnum=randint((ll)1,x-1);
		ll tur=0,rab=0;
		ll p=1,q;//配合记录GCD的乘积.
		do
		{
			for(int i=1;i<=128;i++)
			{
				tur=func(tur,rndnum,x),//乌龟跑一步.
                rab=func(func(rab,rndnum,x),rndnum,x);//兔子跑两步.
				q=(__int128_t)p*abs(tur-rab)%x;//求积.
				if(tur==rab||q==0)
                //tur==rab, 那么找到环; q==0, 那么积为0, 不产生贡献, 跳出.
				{
					break;
				}
				p=q;
			}
			ll _gcd=gcd(p,x);
			if(_gcd>1)
			{
				return _gcd; 
			}
		}while(tur!=rab);
	}
}
